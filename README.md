# Quick Notes

Quick Notes is a minimal and fast note-taking application designed for quick thoughts, meeting notes, and temporary writing without requiring any login or account. It works entirely in the browser using localStorage, allowing users to type freely and download their notes as a `.txt` file when finished. The application functions both online and offline, providing a simple and distraction-free writing experience.

---

## Features

- Clean and minimal interface
- Automatically stores notes in the browser using localStorage
- Download notes as a `.txt` file
- No login or account needed
- Works offline
- Lightweight React application

---

## Technologies Used

- React  
- Vite  
- TailwindCSS  
- localStorage
